# EXPLORATION_04 : 작사가 인공지능 만들기

## 학습 목표
- 시퀀스 데이터 이해
- 텍스트 데이터 토큰화
- tf.keras.Model을 Subclassing

## 진행 과정
- Step 1. 데이터 다운로드
- Step 2. 데이터 읽어오기
- Step 3. 데이터 정제

- Step 5. 모델 설계하기
